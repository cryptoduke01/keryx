# Hackathon Materials (Internal)

This folder contains materials used to generate realistic demo/traction data for the Lepton Agents Hackathon submission.

## Contents
- `traction-run.sh` — Script that fires batches of real x402 calls (mix of seeded tools + external creator payouts). Run **locally only**.
- `SUBMISSION.md` — Shot list, voice-over script, and Google form answers for the video submission.
- `FEEDBACK_TO_CLAUDE.md` — Internal retro notes.

## Important
- These are **not** part of the core Kēryx product experience or claims.
- The actual product supports real paid calls via the x402 protocol + USDC on Arc (see main docs and `/live`).
- MCP endpoints for agents (Cursor, Claude, etc.) currently run in subsidized/demo mode because the MCP spec/clients do not yet carry payment primitives. Direct API usage is the paid path.
- Any volume generated by these scripts is for demo purposes only and is clearly disclosed.

Use these locally when you want `/live` to look active for recording or judging.

Do not treat the existence of this folder as evidence of misleading marketing.
